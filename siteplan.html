01

Overview

The World e-Travel Explorer website provides comprehensive information about any country in the world for travelers and adventurers planning their next journey.

The Problem

Individual travelers must visit many different websites and blogs to research a country before visiting. There is no single, all-in-one reference covering culture, geography, weather, and key facts in one place.

The Motivation

A personal dream of travel inspired this project — building an interactive app where users of any age can explore any country in the world without bouncing between multiple websites. It combines a passion for geography, travel, foreign culture, and real-life data that is useful to everyone who loves to travel.

03

Major Functions

🗺️
1

Country Detail View

Select a country to open a full detail page displaying its flag, capital, population, current weather, languages, and currencies.

🔍
2

Country Search

Users type a country name and the app calls the REST Countries API, rendering results as card-style layouts on the home page.

🌤️
3

Weather Widget

Open-Meteo API fetches live capital city weather including temperature (°C/°F), wind speed, and a visual condition icon.

📍
4

Points of Interest

Geoapify Places API fetches detailed information about attractions, restaurants, and accommodations at the destination.

🌡️
5

°C / °F Toggle

Users toggle between Celsius and Fahrenheit when viewing weather data; preference saved in localStorage.

🌐
6

Filter by Region

Browse countries filtered by seven regions: Africa, Asia, Europe, North America, South America, Oceania, and Antarctica.

🏛️
7

Attraction Detail Cards

Cards display attraction name, category, and a short description fetched from the OpenTripMap place detail endpoint.

❤️
8

Favorites Page

A dedicated page where users can view all countries they have saved as favorites to visit later.

🔗
9

Border Country Navigation

Neighboring countries are displayed as clickable chips — tap one to jump directly to that country's detail page.

🌙
10

Dark / Light Mode Toggle

A toggle switches between dark and light themes; user preference is saved in localStorage.

⚠️
11

Error Handling

Handles all API errors gracefully. If the main country data fails, a clear error message with a retry button appears. If only weather or attractions fail, a small warning appears in that section while the rest of the page continues to work normally.

05

External Data Sources

API 1

REST Countries API

Provides country facts: name, flag, capital, population, region, languages, currencies, border countries, and lat/lng coordinates.

namecapitalregionsubregionpopulation lat/lngareaflagscurrencieslanguages borderstimezonesmaps
API 2

Open-Meteo

Provides current weather for capital cities including temperature, wind speed, and weather condition codes.

temperaturewindspeedwinddirectionweathercode humidityprecipitation_probabilityvisibilityuv_index
API 3

Geoapify Places API

Returns nearby attractions, restaurants, and accommodations with names, categories, and addresses.

namecountrycitycategories address_line1address_line2latlondistance
localStorage: Favorite countries list · Temperature unit preference (°C / °F) · Dark/light mode preference
